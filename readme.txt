Title : Large Integer library for +,-,*,/,% operations. Tested with 256 bit numbers

Platform:
    1. Code has been developed and tested on Ubuntu 18.04 LTS
    2. GCC version 7.5

Installation Instructions:
    sudo apt-get install gcc

Build Command
    gcc -g test_large_num.c large_integer.c -o large_int_demo

Run Command
    ./large_int_demo

Demo Details:
    The demo covers 5 tests each for addition, subtraction, multiplication, division, and modulus for Large Integer
